CSR WEBSITE FEATURES - IMPLEMENTATION PROMPTS

1. CSR Activities Map (Location-based)
--------------------------------------------

Purpose:
Show where CSR initiatives are happening geographically.

Data Structure:
[
  {
    "title": "Tree Plantation Drive",
    "location": "Hyderabad",
    "coordinates": { "lat": 17.3850, "lng": 78.4867 },
    "category": "Environment",
    "impact": "500 trees planted"
  }
]

UI Structure:
- Section Title: "Our CSR Presence Across India"
- Full-width interactive map
- Markers for each activity
- Popup on click with:
  - Title
  - Location
  - Category
  - Impact

Placement:
After Hero section OR Impact section

CLI Prompt:
Create a CSR Activities Map section using React and Leaflet.js.
Requirements:
- Display a full-width interactive map
- Add markers for CSR activities using latitude and longitude
- On clicking a marker, show a popup with:
  - Title
  - Location
  - Category
  - Impact
- Use mock JSON data
- Ensure responsiveness
- Section title: "Our CSR Presence Across India"
